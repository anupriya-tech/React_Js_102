Que1) Which link will open 'google' and why?

   <a href="https://www.google.com/" onclick="console.log('clicked');">Link One</a>
   <a href="https://www.google.com/" onclick="console.log('clicked'); return false">Link two</a> 
   
   Answer: The first link
 will open google browser Because using return false The return false will return false to the click-event. That tells the browser to stop following events, like follow a link.

Que2) var student = {
     name: "John",
     getName: function()
     {
         console.log("student " + this.name);
     }
   }
   var employee = {
     name: "Tom",
     getName: function()
     {
         console.log("employee " + this.name);
     }
   }
   var run = student.getName.bind(employee);
   run();
  

Answer: output 'student Tom'  that allow a function to be invoked as if it were a method of another object.
